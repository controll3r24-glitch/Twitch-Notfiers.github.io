📜 Termini di Servizio

Ultimo aggiornamento: 25 Aprile 2026

1. Accettazione dei Termini

Utilizzando il bot Discord Controll3r24 Live Notifier (d'ora in poi "il Bot"), accetti integralmente questi Termini di Servizio. Se non sei d'accordo con una qualsiasi parte, ti preghiamo di non utilizzare il Bot.

2. Descrizione del Servizio

Il Bot fornisce notifiche automatiche relative agli streaming live del canale Twitch Controll3r24. Le funzionalità includono:

  • Rilevamento automatico della messa in onda su Twitch.
  • Invio di messaggi (embed) nei canali Discord designati.
  • Possibilità di menzionare un ruolo configurabile.
  • Comandi come !twitch per ottenere il ruolo di notifica, e pannello di configurazione !panel per gli amministratori.

3. Utilizzo delle API di Twitch

Il Bot si interfaccia con le API ufficiali di Twitch per verificare lo stato della diretta. Non raccogliamo né conserviamo dati personali degli utenti di Twitch. L'uso delle API è conforme ai Termini di Servizio di Twitch.

4. Privacy e Dati

Il Bot non raccoglie, memorizza o condivide dati personali degli utenti. Le uniche informazioni salvate sono:

  • ID del canale Discord dove inviare le notifiche.
  • ID del ruolo da menzionare.
  • Preferenze di personalizzazione del messaggio (JSON fornito dagli amministratori).

Nessuno di questi dati è riconducibile a singoli individui. Non vengono registrati log dei messaggi né cronologia degli utenti.

5. Responsabilità dell'Utente

Gli amministratori del server Discord che utilizzano il Bot sono responsabili di:

  • Configurare il Bot in conformità con le regole del proprio server e con i Termini di Servizio di Discord.
  • Non utilizzare il Bot per inviare spam, contenuti offensivi o illegali tramite il messaggio personalizzato.
  • Assicurarsi che il ruolo menzionato non violi la privacy degli utenti o causi disturbo.

6. Limitazione di Responsabilità

Il Bot viene fornito "così com'è", senza garanzie di alcun tipo. Non ci assumiamo responsabilità per:

  • Eventuali interruzioni del servizio dovute a problemi tecnici o cambiamenti delle API di Twitch/Discord.
  • Danni diretti o indiretti derivanti dall'uso o dall'impossibilità di usare il Bot.
  • Contenuti trasmessi attraverso il messaggio live personalizzato dagli amministratori.

7. Modifiche ai Termini

Ci riserviamo il diritto di aggiornare questi Termini di Servizio in qualsiasi momento. Le modifiche saranno pubblicate in questa pagina. L'uso continuato del Bot dopo le modifiche costituisce accettazione dei nuovi termini.

8. Contatti

Per domande, segnalazioni o chiarimenti, puoi contattarci attraverso il server Discord del creatore o tramite il canale Twitch Controll3r24.
